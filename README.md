A simple Chess Game Developed Completely using java language, this game implement the basic functions like: 
1. pieces movement 
2. checks and checkmate 
3. Pins 
4. stalemate 
5. pawn promotion to (Queen, Knight, Bishop or Rook)
6. illegal Moves 
7. castling
